#### 3. SQL 'JOIN'

```SQL
SELECT c.customerID, c.CustomerName, o.OrderID, o.OrderDate
FROM Customers as c
	LEFT JOIN Orders AS o ON c.customerID = o.customerID
WHERE o.OrderID is null
```
